# Paises y Monedas

Los códigos de país están en formato ISO 3166-1 alpha-2. Las monedas están en formato ISO 4217.

### Detalle

| País | Código del país (ISO 3166-1 alpha-2) | Código de moneda (ISO 4217) |
| -------- | ------- | ------- |
| Chile | CL | USD / CLP |
| Perú  | PE | USD / PEN |
| Uruguay | UY| USD / UYU |
| Paraguay | PY | USD / PYG |
| Brazil | 	BR | USD / BRL |
| Ecuador | EC | USD |
| México | MX | USD / MXN |
