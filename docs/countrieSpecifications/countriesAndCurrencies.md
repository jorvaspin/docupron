# Countries and currencies

Country codes are in ISO 3166-1 alpha-2 format. The coins are in ISO 4217 format.

### Details

| Country | Country code (ISO 3166-1 alpha-2) | Currency code (ISO 4217) |
| -------- | ------- | ------- |
| Chile | CL | USD / CLP |
| Perú  | PE | USD / PEN |
| Uruguay | UY| USD / UYU |
| Paraguay | PY | USD / PYG |
| Brazil | 	BR | USD / BRL |
| Ecuador | EC | USD |
| México | MX | USD / MXN |
